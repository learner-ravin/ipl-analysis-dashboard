<h1>IPL Data Analysis Dashboard</h1>

<h2>Overview</h2>

<p>
  This Power BI project presents an end-to-end analysis of Indian Premier League (IPL) data of all 18 seasons from 2008 to 2025. 
  It transforms raw match and player data into meaningful insights through interactive visualizations, enabling a clear understanding of team performance, player statistics, and match outcomes.
</p>

## Dashboard Sections

The dashboard is structured into **five main sections**, each focusing on a different aspect of IPL analysis.

---

### 🔹 1. Overview Analysis

This section provides a high-level summary of the IPL dataset using key metrics and aggregated insights.

* Displays total matches, teams, runs, wickets, and venues
* Shows match outcome distribution (batting first vs second)
* Bar graph of title distribution across teams
